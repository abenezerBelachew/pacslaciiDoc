Overview


The PACSLAC II Checklist is used to screen for pain in older adults who have dementia or other cognitive impairment with a limited ability to communicate. The tool should be used by a professional nurse. We will be visualizing the data we receive from this assessments in a user-friendly way so users of our project can better understand their patients.

PACSLAC II - Pain Assessment Checklist for Seniors with Limited Ability to Communicate II
User - User can be the patient, caretaker or the person responsible for conducting research on pain levels across different groups.
Patients - Seniors with limited ability to communicate
Heatmap - representation of data in form of a map in which data values are represented as colours
Heatmap calendar - A simple line graph, where the horizontal axis plots the time and the vertical axis plots the measured variable. The data is displayed in a format of heatmap.

      Acceptance Tests: Indicator with available view options is evident. The map displays province view if the province view is selected. Similar for every other view offered. The map zooms into the selected region.

      Acceptance Tests: The correct mode “heat” level is displayed. The time of the displayed map is clearly displayed on the screen. The map shows the country.
