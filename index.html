CHARTIST.JS

Create responsive, scalable and good looking charts with chartist.js. Chartlist.js is a javascript library that helps build responsive charts.

Population Pyramid

This diagram shows the distribution of age groups in the United States over the last 150 years. Use the arrow keys to observe the changing population.

Calendar View

This chart shows daily changes of the Dow Jones Industrial Average from 1990 to 2010. Days the index went up are green; days the index went down are pink.

Hierarchical Bar Chart

Click a blue bar to drill down, or click the background to go back up. This bar graph is made using the d3.js library available online.

Google Calendar like Display

A simple Google calendar like display to visualize temporal data. Use the forward and backwards buttons to move through the months.

Cubism.js

Cubism.js is a D3 plugin for visualizing time series. Use Cubism to construct better realtime dashboards, pulling data from Graphite, Cube and other sources.

Choropleth

Choropleth makes use of the famous javascript library, d3.js for visualizing heatmaps. Unemployment rate by county, August 2016. Data: Bureau of Labor Statistics.

An Integrated Version of Security Monitor Vis System
